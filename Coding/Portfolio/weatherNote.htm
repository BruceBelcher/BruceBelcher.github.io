Weather Forecast Application

Written using node.JS, (JavaScript on the server) and express, (to build web applications and communicate through browser).

Utilises 2 API calls, 1st to mapbox which returns the lat and long of the location specified and 2nd to darksky to get the forecast for that location.

It's a node.JS application so 'hosted' on heroku in order to run it from my Portfolio.
To run locally, cd to the application directory and call 'node app.js'. Result on localhost:3002.
To run publicly, from the Portfolio web site, call https://afternoon-stream-66915.herokuapp.com/
To upgrade and upload to heroku; cd to application directory and make changes. git add . git commit -m "message" git push.
